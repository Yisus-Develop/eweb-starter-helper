# EWEB - Starter Helper

Essential initial setup for WordPress projects. This plugin automates common tasks like safe SVG support, Elementor cleanup, and performance optimizations.

## Features

- **Modular System**: Toggle only the tools you need.
- **Dynamic Copyright**: Shortcodes and Elementor widget that update themselves.
- **Universal Duplicator**: Clone any post type or Elementor template with one click.
- **GitHub Auto-Updates**: Receive the latest fixes and features directly in your WordPress dashboard.
- **Professional i18n**: Fully translatable (includes .POT file).

## Installation

1. Download the latest [Release](https://github.com/Yisus-Develop/eweb-starter-helper/releases).
2. Upload the `.zip` to your WordPress site.
3. Activate and configure in **EWEB Helper**.
