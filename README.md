The professional "Swiss Army Knife" for WordPress development. This modular toolkit provides high-performance utilities, security hardening, and development boosters for modern WordPress projects. Part of the **EWEB Plugin Suite**.

## 🚀 Modular Power

Enable only what you need to keep your site fast and lean:

- **SVG Manager:** Secure SVG uploads with automatic sanitization.
- **Elementor Optimization:** Clean up bloated scripts and styles for better LCP scores.
- **Security Suite:** Protect the REST API, hide login errors, and disable XML-RPC.
- **Performance Engine:** Disable unnecessary WordPress features (Emojis, Embeds, RSD, etc.).
- **Smart Duplicator:** Clone posts and pages with a single click.
- **Custom Widgets:** Professional widgets for Elementor (Copyright, dynamic text, etc.).

## 🏗️ Technical Highlights

- **Singleton Pattern:** Clean, efficient initialization to avoid memory leaks.
- **Modular Settings:** User-friendly dashboard to manage core features.
- **GitHub Sync:** Built-in system for seamless updates directly from GitHub.
- **Clean Code:** Strictly typed components and namespaced architecture.

## 🛠️ Installation

1. Upload the `eweb-starter-helper` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Access the **Starter Helper** settings to activate your modules.

## 🔄 Updates

This Pro edition supports automatic updates directly from GitHub. Repository slug: `Yisus-Develop/eweb-starter-helper`.
